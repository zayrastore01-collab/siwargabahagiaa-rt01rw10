# siwargabahagiaa-rt01rw10
Selamat datang di Portal Warga RT 01 RW 10. Website ini dibuat untuk mempermudah pelayanan administrasi warga secara digital, meningkatkan transparansi pengelolaan kas RT, mempermudah pelaporan keluhan, absensi kegiatan warga, serta mempercepat komunikasi antara pengurus RT dan warga tanpa harus datang langsung ke rumah pengurus.
